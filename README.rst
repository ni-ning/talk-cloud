安装
********


为了简化安装过程，推荐使用 pip 进行安装

.. code-block:: bash

    pip install talk-cloud

升级 talk-cloud 到新版本::

    pip install -U talk-cloud

如果需要安装 GitHub 上最新代码::

    pip install https://github.com/ni-ning/talk-cloud/archive/master.zip
